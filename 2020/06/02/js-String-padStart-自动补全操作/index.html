发表于 2020-06-02 更新于 2025-03-24
字符串自动补全函数 注意一定是字符串！

padStart(targetLength [, padString]) 方法用另一个字符串填充当前字符串(重复，如果需要的话)，以便产生的字符串达到给定的长度。填充从当前字符串的开始(左侧)应用的。
targetLengt 目标长度
padString 补充的字符串
