vue2.0 组件复用 数据没有更新

发表于 2020-06-02 更新于 2025-03-24

如果开发过程中出现使用v-if-else来切换组件时发现数据没有更新，那么就是因为元素被复用具体可以参考vue官网-组件

查看input元素发现只有input的placeholder的变化了 元素本身就没有改变，而且输入的值也没有被清除；原因是vue为了尽快的渲染页面所以通常会复用已经渲染的元素，所以会导致绑定的数据并没被实时刷新；
解决办法就是：对复用的元素添加 key

这样就避免了数据没有刷新的问题
