git 提交规范

规范的优势

  1. 版本回退可以快速定位到指定版本
  2. 清晰明了知道每次提交的内容
  3. 统一规范

commit 提交格式
commit 提交包括三个部分 Header, Body和 Footer;
其中 Header 是必须的, Body和Footer是可忽略。
